# Disk Cleanup Skill

## Description
This skill is used to clean various temporary files, caches, and system junk on the C drive to free up disk space. The script requires administrator privileges to run properly.

## Features
- Clean system temporary files
- Empty recycle bin
- Clear browser cache (Chrome, Edge, Firefox)
- Clean Windows update files
- Clean system log files
- Clean Windows Defender temporary files
- Clean volume shadow copies

## Usage
1. Run this script with administrator privileges
2. The script will automatically perform all cleaning tasks
3. Completion information will be displayed upon finish

## Notes
- This script requires administrator privileges
- Ensure no important temporary files need to be retained before running
- Recommend running regularly to maintain system performance
